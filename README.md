# LifeTrak

A personal iOS app for tracking daily habits — water intake, medication, exercise, journaling, and more. Built with SwiftUI and SwiftData, targeting iOS 26.2.

## Status

In early development using a **tracer round** strategy: building water tracking end-to-end before generalizing to other tracker types. The water tracer round is complete. Next up: Apple Health integration (sleep tracking).

## Features (Water Tracer Round)

- **Quick logging** — one-tap to record water intake with haptic feedback
- **Today screen** — progress ring, current total vs. daily goal
- **History** — browse past entries grouped by day; edit, delete, or backfill entries
- **Progress visualization** — animated progress ring, weekly bar chart, streak tracking, goal celebration
- **Settings** — configurable daily goal and default serving size
